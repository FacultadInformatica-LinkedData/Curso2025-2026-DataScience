Descripción de la aplicación

Nuestra aplicación tiene como objetivo representar la estructura del Sistema de Transporte Colectivo Metro de la Ciudad de México utilizando datos abiertos y técnicas de Linked Data. A partir de la información de líneas, estaciones, afluencia diaria y metadatos geográficos, construiremos un modelo semántico que permitirá:

  • Visualizar la red de metro
  • Consultar información enriquecida sobre cada estación (fecha de inauguración, alcaldía, municipio)
  • Analizar patrones de movilidad y afluencia

Posteriormente, esta base de conocimiento se utilizará como fuente de datos para nuestro proyecto de la asignatura "Inteligencia Artificial", donde aplicaremos heurísticas de búsqueda para hallar caminos mínimos sobre la red utilizando tiempos reales obtenidos mediante la API abierta de transporte de la CDMX.

Funcionalidades principales

  • Visualización interactiva de la red de metro
  • Análisis de la afluencia por línea y estación, con filtros por fecha.
  • Consultas semánticas mediante SPARQL
  • Integración de Linked Data

Uso y manipulación de los datasets

Se procesarán y transformarán los datasets "Líneas y Estaciones" y "Afluencia Diaria" para que sirvan como base del proyecto. No obstante, planeamos añadir información complementaria para enriquecer los datos.
